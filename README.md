cd byword-project
# ByWord Project


Projeto com módulo inicial de geração de caça-palavras (word search) com backend em Django e frontend em Vue.

## 📌 Módulo atual

Este módulo permite:

- Criar WordSearch (nome, tamanho do grid)
- Adicionar palavras
- Gerar automaticamente o grid
- Exportar:
  - PDF (com solução)
  - PNG (com e sem solução)
- Download direto pelo Django Admin
